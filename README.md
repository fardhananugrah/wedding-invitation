# Undangan Pernikahan Digital

Aplikasi web untuk membuat dan membagikan undangan pernikahan digital yang elegan dan interaktif.

## Fitur

- 🎨 Desain modern dan responsif
- 💌 Mudah dibagikan melalui link
- 📱 Kompatibel dengan semua perangkat
- ✨ Animasi yang indah
- 🎵 Dukungan musik latar

## Teknologi

- React 18
- React Router
- CSS3
- JavaScript ES6+

## Instalasi

```bash
# Clone repository
git clone https://github.com/fardhananugrah/wedding-invitation.git

# Masuk ke direktori
cd wedding-invitation

# Install dependencies
npm install

# Jalankan aplikasi
npm start
```

Aplikasi akan berjalan di `http://localhost:3000`

## Build untuk Production

```bash
npm run build
```
